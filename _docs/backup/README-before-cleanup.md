# About

I am an undergraduate student in the Division of ICT Convergence at Hanyang University ERICA, with a double major in Convergence Security. I am expected to graduate in **February 2027**.

I am currently working as an undergraduate research student at **ISS Lab, Korea University**, where I study security-related topics with a focus on Android security, system security, and software security analysis.

My current interests include Android application security, system security, cloud and platform engineering, and security-oriented static and dynamic analysis. I am particularly interested in understanding how real-world systems enforce security decisions across application, framework, and platform boundaries.
